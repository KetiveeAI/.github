## About

Ketivee is an open-source software development organization building tools that are accessible to everyone — from first-time contributors to seasoned engineers. We believe great software is built in the open, together.

Our community is welcoming, collaborative, and always growing. Whether you're here to contribute code, improve documentation, report bugs, or share ideas, there's a place for you at Ketivee.

## Getting Started

New to Ketivee? Here's how to jump in:

1. **Explore our repositories** — browse the projects above and find one that interests you.
2. **Read the contribution guidelines** — review [CONTRIBUTING.md](CONTRIBUTING.md) before submitting anything.
3. **Pick up an issue** — start with a [Good First Issue](https://github.com/ketiveeai/ketivee-docs/issues?q=is%3Aopen+is%3Aissue+label%3A%22good+first+issue%22) if you're new.
4. **Join the community** — hop into our [Discord](https://blog.ketivee.com) to introduce yourself and ask questions.
